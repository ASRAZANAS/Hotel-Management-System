

Hotel Management System 🏨
A simple and interactive console-based application to manage hotel room records, built using Python. This project was developed to practice core programming concepts such as data structures and control flow.

🛠 Tech Stack
Language: Python

Data Structures: Lists, Dictionaries

Programming Concepts: * Loops (while, for)

Conditionals (if-elif-else)

Functions (def)

User I/O (input(), print())

Environment: Jupyter Notebook / Google Colab
